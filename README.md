# ASM Asset Discovery Service

Backend microservice for the Attack Surface Management (ASM) platform, responsible for
domain management and passive DNS asset discovery (A, AAAA, NS, MX records).

**Status:** work in progress — built incrementally, one commit per milestone. See commit
history for the build order. Full setup, architecture, and design-decision documentation
will land in the README once the implementation is complete.

## Requirements

- Python 3.12+
